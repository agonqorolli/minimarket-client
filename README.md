# MiniMarket Client
React Implementation of MiniMarket App.
Serverless Infrastructure of this App can be found in this [repo](https://github.com/agonqorolli/minimarket-server),
and live hosting of such app can be found [here](https://main.d2jsglc8ny66a.amplifyapp.com/).

---

### How to test it

1. Create `.env` with corresponding variables. See `.env.example` for a reference.
    1. Already provided Stripe's publishable test key for quicker testings.
1. Run `npm install` to install node packages.
1. Run `npm start` to start the server locally.
1. Navigate to ` http://localhost:3000/` to test the App.
